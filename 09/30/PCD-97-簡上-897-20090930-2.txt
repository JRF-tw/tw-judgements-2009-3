臺灣板橋地方法院刑事判決             97年度簡上字第897號
上  訴  人
即  被  告  甲○○
選任辯護人  蔡明和律師
上列上訴人因詐欺案件，不服本院97年度簡字第5634號中華民國
97年6 月26日第一審判決（聲請簡易判決處刑案號：97年度偵字
第12874 號），提起上訴，本院管轄之第二審合議庭判決如下：
    主  文
上訴駁回。
    事實及理由
一、本案經本院審理結果，認第一審簡易判決認被告甲○○犯刑
    法第30條、第339 條第1 項之幫助詐欺取財罪，判處拘役五
    十日，如易科罰金以新臺幣一千元折算一日，認事用法及量
    刑均無不當，應予維持，除犯罪事實部分，應予補充更正「
    被告係於民國96年9 月20日，在臺北縣泰山鄉某處，將其本
    件華南商業銀行（下簡稱「華南銀行」）泰山分行帳戶之提
    款卡及密碼（聲請書誤贅載尚有該帳戶之存摺，應予更正）
    ，提供予姓名年籍不詳之人，供作詐欺集團作為詐騙被害人
    匯款之人頭帳戶使用，嗣即有己○○（被害詐騙地點係在臺
    北縣中和市）、丁○○（被害詐騙地點係在新竹縣竹北市）
    、丙○○（被害詐騙地點係在臺南縣永康市）等人遭詐騙集
    團成員詐騙匯款至被告上開銀行帳戶，提領一空」等情外，
    其餘均引用如附件第一審判決書記載之事實、證據及理由。
二、被告上訴意旨略以：其本件銀行帳戶提款卡及密碼，純係因
    遺失而遭詐欺集團使用，蓋其使用上開銀行帳戶作為薪資轉
    帳已有四年以上，並作為其弟保險費扣款帳戶亦有二年以上
    ，顯見其該帳戶正常長期使用，豈有可能任意提供詐欺集團
    作為人頭帳戶不法使用，況其有正當工作，收入固定，生活
    正常，並無負債，經濟無虞，亦無犯罪之動機，此外其發現
    遺失後即向警方報案，經警調閱提款機錄影資料，亦發現係
    一名不詳之成年男子持被告遺失之提款卡提領被害人匯款，
    被告並無幫助詐欺之犯罪意圖，爰提起上訴，請撤銷原判決
    ，改判被告無罪云云，辯護人亦為被告辯護同上意旨。
三、但查，被告上開犯罪事實，業據告訴人己○○、被害人丁○
    ○、丙○○等於警詢指證被害情節明確（見偵查卷5 至6 頁
    、9 至10頁、44至46頁，上開告訴人、被害人於警詢之陳述
    ，被告及辯護人於審理時均不爭執其證據能力，依刑事訴訟
    法第159 條之5 規定，均得為證據），並有被告上開華南商
    業銀行泰山分行帳戶開戶資料及交易明細、被害人丙○○被
    騙匯款之中華郵政公司自動櫃員機儲戶交易明細表等附卷可
    資佐證。被告雖辯稱上開意旨云云，然查：
    (一)依被告於警詢及本院審理時所稱：伊最後一次使用本件銀
      行帳戶提款卡，係於96年9 月20日，在臺北縣泰山鄉○○
      路○ 段與福泰街口之「全家便利商店」（經查該處係臺北
      縣泰山鄉○○路○ 段109 號「全家便利商店」泰山泰林店
      ，其內設置之提款機係國泰世華銀行之提款機）提款，領
      完錢後即將提款卡放在伊皮夾內，直至同年月26日下午6
      時許，在泰山鄉○○路○ 段，要去查詢及提領其上班之「
      愛的世界股份有限公司」（下簡稱「愛的世界公司」）發
      下的門市零用金，才發現遺失，當時伊僅有遺失提款卡而
      已，伊想不起來提款卡為何會不見云云（見偵查卷3 頁、
      本院97年11月14日準備程序筆錄2 至3 頁、98年5 月19日
      審判筆錄8 至9 頁、13至15頁），惟據被告上開銀行帳戶
      存款往來明細表所示，其於96年9 月20日係提領新臺幣（
      下同）1900元，提領之交易行代號係「193 」，經核被告
      該提領之款項並無另扣跨行提領之6 元手續費，可見被告
      提領之自動櫃員機係屬其開戶銀行即華南銀行之提款機，
      再核諸上開交易行代號「193 」，經查係華南銀行泰山分
      行（設於臺北縣泰山鄉○○路○ 段99號）之分行代號，此
      外再查核華南銀行在臺北縣設置自動櫃員機據點之便利商
      店，該銀行合作設置提款機之合作便利商店，亦僅有「萊
      爾富」及「OK」等便利商店，並無與「全家便利商店」合
      作設置提款機，此有華南銀行網站所載分行及臺北縣提款
      機設置地點等列印資料在卷可參，足認被告於96年9 月20
      日提款之地點實際上應係在上址之華南銀行泰山分行，而
      非被告上開所稱之上址全家便利商店云云，被告上開所述
      與事實顯有不符，然其卻隱匿事實，另虛構提款地點，其
      動機已有可議，且依被告上開所述其本件遺失之提款卡當
      時使用後係放在其皮夾內，但卻僅有該提款卡遺失，而被
      告亦無法具體說明其該提款卡究係如何遺失，此亦均與常
      理有悖，因此稽之上述諸點，被告上開所辯是否屬實，顯
      非無疑。
    (二)次依被告上開所辯，其之後係於同年月26日欲持該提款卡
      查詢及提領公司發放之該月零用金，始發現該提款卡遺失
      云云，惟復依被告上開華南銀行帳戶存款往來明細表所示
      ，被告於96年9 月份代理愛的世界公司泰山明志店門市店
      長時申報之當月零用金312 元，早於同年月17日即經愛的
      世界公司核發匯款至其本件銀行帳戶內，其並已於96年9
      月20日最後一次提款時，連同於同日匯入之其該月愛的世
      界公司核發應領之業務獎金1653元，一起提領，此亦經證
      人即愛的世界公司督導人員戊○○於本院審理時證稱：被
      告代理店長時申報之門市雜項費用是由伊核可的，統一在
      每月由會計轉帳匯入被告帳戶，再由被告領出放回門市零
      用金款項內，被告本件銀行帳戶96年9 月17日匯入之312
      元，就是門市的雜項費用，同年月20日匯入之1653元，應
      該是獎金等語（見本院98年8 月25日審判筆錄4 至8 頁）
      ，以及證人即愛的世界公司會計庚○○於本院審理時證稱
      ：伊公司零用金之發放一般是固定於每月15日，偶而會因
      假日或資金調度順延，但都會在每月20日前處理完畢，伊
      庭陳之「門市費用--甲○○轉帳明細」（包括96.6.23 至
      96 年10 月15日轉帳匯給被告之門市費用，含96年9 月17
      日轉帳匯款312 元）即匯給被告門市零用金資料等語（見
      本院98年8 月25日審判筆錄11至12頁）明確，足見被告上
      開所辯其之後係於同年月26日欲持該提款卡查詢及提領公
      司發放之該月零用金，始發現該提款卡遺失云云，要與事
      實不符，顯係被告飾卸之詞，應不足採，由此益見被告上
      開所稱其本件銀行提款係遺失云云，顯非事實。
    (三)再被告上開所辯其本件銀行帳戶使用情形及其個人工作、
      經濟等狀況，固非無見，然另稽之被告上開銀行帳戶交易
      明細資料所示，可見詐欺集團成員精準地掌握在被告上開
      帳戶留存之款項7069元，於被告所稱提款卡遺失之次日即
      96年9 月21日經保險公司按期扣款被告應繳納之保險費共
      7000元，僅餘69元後，旋即於同日詐騙上開告訴人、被害
      人等轉帳匯入款項至被告上開帳戶內提領一空，若非知悉
      被告帳戶款項使用情形之人，何能如此精確掌握被告帳戶
      扣款後，始加以作為詐騙被害人匯款使用，其時間之相近
      巧合，於常情顯然有違，已非無疑，況被告所述遺失之情
      果真屬實，拾獲之詐欺集團豈有甘冒掛失不能提款之風險
      ，加以利用。是綜上所述，參互印證，足見被告上開所辯
      ，應係卸責之詞，不足採信，應有交付其上開銀行帳戶提
      款卡及密碼給他人使用之情，再衡諸被告於本件案發時已
      從事工作多年，係有社會經驗之成年人，且使用銀行帳戶
      多年，其當可預見一般人無故不自行申辦金融機構帳戶使
      用，反以價購或其他方式取得他人申辦之金融機構帳戶使
      用，可能為遂行詐欺取財之不法所有意圖，而其仍提供其
      上開銀行帳戶提款卡及密碼，供他人使用，難謂其無幫助
      詐欺之不確定故意可言，是本件被告犯行事證明確，堪以
      認定。
四、綜上所述，原審判決認事用法，核無不合，另衡諸被告無前
    科、本件犯罪動機緣由、手段、對社會交易安全、告訴人、
    被害人等所生危害及犯後辯飾態度等一切情狀，原審判決量
    刑亦屬妥適。上訴意旨仍執前詞指摘原審判決不當，求予撤
    銷改判，顯無理由，自應予駁回。
據上論斷，應依刑事訴訟法第455 條之1 第1 項、第3 項、第36
8 條、第373 條，判決如主文。

中華民國刑法第30條
（幫助犯及其處罰）
幫助他人實行犯罪行為者，為幫助犯。雖他人不知幫助之情者，
亦同。
幫助犯之處罰，得按正犯之刑減輕之。
中華民國刑法第339條
（普通詐欺罪）
意圖為自己或第三人不法之所有，以詐術使人將本人或第三人之
物交付者，處5年以下有期徒刑、拘役或科或併科1千元以下罰金
。
以前項方法得財產上不法之利益或使第三人得之者，亦同。
前二項之未遂犯罰之。

│  (一)上揭犯罪事實，業據被害人己○○、丁○○及丙○○於警詢          │

│    細、丙○○之中華郵政公司自動櫃員機儲戶交易明細表1 紙          │
│    在卷可稽。是前開帳戶確已遭詐欺集團用以當作詐騙被害人          │
│    所得之匯款帳戶甚明。                                          │
│  (二)次查，金融帳戶之存摺、金融卡乃個人理財之重要工具，一          │
│    旦遺失，一般人均會向警方報案或向開戶銀行申請掛失，然          │
│    被告知悉該等物品遺失後，卻未立刻報警處理，已有可議之          │
│    處。再衡諸一般人使用金融帳戶提款之習慣，為避免遺失存          │
│    摺或提款卡時，帳戶內之存款遭人盜領，通常均將提款卡、          │
│    存摺與密碼分別存放，而不至於將密碼記載於提款卡上或與          │
│    提款卡同時存放，否則密碼之設定即失其意義；再另自詐騙          │
│    集團之角度審酌，渠等既知利用他人之帳戶掩飾犯罪所得，          │
│    應非愚昧之人，當知社會上一般正常之人如帳戶存摺、提款          │
│    卡、印鑑遭竊或遺失，為防止拾得或竊得之人盜領其存款或          │
│    做為不法使用而徒增訟累，必於發現後立即報警或向金融機          │
│    構辦理掛失止付，在此情形下，如仍以該帳戶作為犯罪工具          │
│    ，則在渠等向他人詐騙，並誘使被害人將款項匯入該帳戶後          │
│    ，極有可能因帳戶所有人掛失止付而無法提領，則渠等大費          │
│    周章從事於犯罪之行為，甘冒犯罪後遭追訴、處罰之風險，          │
│    卻只能平白無故替原帳戶所有人匯入金錢，而無法得償犯罪          │
│    之目的，是以犯罪集團若非確定該帳戶所有人不會報警或掛          │
│    失止付，以確定渠等能自由使用該帳戶提款、轉帳，當不至          │
│    於以該帳戶從事犯罪。被告所辯上揭情詞顯與常情有違，是          │
│    系爭帳戶存摺、提款卡及密碼應係被告提供予詐騙集團使用          │
│    。                                                            │
│  (三)再按刑法上之故意，可分為直接故意與不確定故意（間接故          │
│    意），所謂「不確定故意」，係指行為人對於構成犯罪之事          │
│    實，預見其發生而其發生並不違背其本意者，刑法第13條第          │
│    2 項定有明文。查金融帳戶之申請甚為簡易方便，亦無信用          │
│    問題，如有使用帳戶之需要，自可自行開戶，如非供作犯罪          │
│    出入之帳戶或其他不法目的，應無隱匿自己名義帳戶而使用          │
│    他人金融帳戶之必要，且邇來利用刮刮樂、退稅、求職等名          │
│    義詐欺取財之犯罪類型層出不窮，該等犯罪，多數均係利用          │
│    人頭帳戶作為出入帳戶，並經媒體廣為披露，被告係成年且          │
│    智力成熟之人，對此應知之甚詳，竟仍將系爭帳戶存摺、提          │
│    款卡及密碼提供給不詳年籍之人使用，且事後並企圖以遺失          │
│    云云卸責，被告有容任該犯罪集團利用其帳戶掩飾自己犯詐          │
│    欺財物之不確定故意，被告犯嫌應堪認定。                        │
│二、核被告所為，係犯刑法第30條第1 項、第339 條第1 項之幫          │
│    助詐欺取財罪嫌。                                              │
│三、依刑事訴訟法第451 條第1 項聲請逕以簡易判決處刑。              │
