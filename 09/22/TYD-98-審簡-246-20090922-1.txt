臺灣桃園地方法院刑事簡易判決         98年度審簡字第246號
公  訴  人  臺灣桃園地方法院檢察署檢察官
被      告  甲○○
上列被告因違反著作權法等案件，經檢察官提起公訴（97年度偵
字第16965 、23576 號），被告自白犯罪，本院認為宜以簡易判
決處刑，爰不經通常審判程序，逕以簡易判決處刑如下：
    主  文
甲○○意圖銷售而擅自以重製於光碟之方法侵害他人之著作財產
權，處有期徒刑玖月。緩刑參年，並向被害人奕鴻文教事業有限
公司支付新臺幣參拾參萬元之財產上損害賠償，給付方式為自民
國九十八年七月五日起至全部清償完畢為止，於每月五日支付新
臺幣壹萬伍仟元予被害人奕鴻文教事業有限公司，如有一期不履
行，視為全部到期。
    事實及理由
一、甲○○明知「奕鴻」、「E HOME」、「HOME RUN ENGLISH」
    、「在家run 英文」等商標名稱及圖樣，係奕鴻文教事業有
    限公司（下稱奕鴻公司）向經濟部智慧財產局（下稱智財局
    ）申請註冊登記而取得指定使用於講義、考試卷、書籍、影
    片、錄影片、影碟片等商品之商標專用權，且經公示於智財
    局商標資料檢索服務網頁，現仍在專用期間內，任何人未經
    上開商標專用權人之同意或授權，不得於同一商品或類似商
    品，使用相同或近似之註冊商標，亦不得販賣或意圖販賣而
    陳列該商標商品；亦明知「謝孟媛英文教學DVD ，Home Run
    English ，在家run 英文，中級英文文法」（全套含光碟片
    120 片，教材6 本）、「謝孟媛英文教學DVD ，Home Run
    English ，在家run 英文，發音篇」（全套含光碟片10片，
    教材1 本）及「謝孟媛英文教學DVD ，Home Run English，
    在家run 英文，初級英文文法」（全套含光碟片60片，教材
    3 本），係奕鴻公司依法取得專屬授權享有重製權、公開發
    行權、公開播映權等著作財產權之視聽及語文著作，非經著
    作財產權人之同意或授權，不得擅自重製上開視聽及語文著
    作。詎甲○○竟基於意圖銷售而以重製方法侵害他人著作財
    產權、意圖銷售而擅自以重製於光碟之方法侵害他人著作財
    產權，及未經商標權人同意，於同一商品使用相同之註冊商
    標之犯意，於民國95年年底某日起，在其桃園縣中壢市○○
    ○街70巷15號2 樓住處，藉由電腦設備及網際網路分別連線
    至YAHOO 奇摩拍賣網站、PChome露天拍賣網站，並各以其申
    設之「mm66168 」、「pp168w」帳號分別登入該等拍賣網站
    ，張貼販賣上開重製且仿冒前揭商標之「謝孟媛英文講義」
    、「謝孟媛Home Run English初級文法」等商品著作物之訊
    息，以每套新臺幣（下同）2,500 元至5,500 元不等之價格
    ，供不特定多數人上網下標選購，並於買家網路標購後，前
    往不詳之影印店重製上開語文著作，以及在其上開住處自行
    以燒錄方式重製前揭視聽著作，完成後再郵寄其重製仿冒前
    開商標之「謝孟媛英文講義」（即前揭「謝孟媛英文教學DV
    D ，Home Run English，在家run 英文，中級英文文法」《
    全套含光碟片120 片，教材6 本》及「謝孟媛英文教學DVD
    ，Home Run English，在家run 英文，發音篇」《全套含光
    碟片10片，教材1 本》）、「謝孟媛Home Run English初級
    文法（即前開「謝孟媛英文教學DVD ，Home Run English，
    在家run 英文，初級英文文法」《全套含光碟片60片，教材
    3 本》）等視聽及語文著作商品交予買家。嗣經奕鴻公司人
    員於網路上發現上開訊息，而提出本件告訴。
二、上開犯罪事實，業據被告甲○○迭於警詢、檢察官訊問及本
    院審理時坦承不諱，並與證人即行動電話門號0000000000號
    之申請人林祥明、IP位址59.104.145.193之申請人莊淑琇分
    別於警詢及檢察官訊問時證述之情節互核相符，復有「奕鴻
    」、「E HOME」、「HOME RUN ENGLISH」、「在家run 英文
    」等商標之智財局商標資料檢索服務網頁列印書面、著作權
    授權合約書（中級英文文法）、著作權授權合約書（發音篇
    ）、鑑定書（英文中級文法及發音篇）、被告於YAHOO 奇摩
    拍賣網站刊登販售謝孟媛英文講義之頁面及評價紀錄等網頁
    列印資料、被告使用之IP位址59.104.145.1 93 使用者資料
    查詢、IP/ASN查詢、中華電信數據CRIS查詢單結果檔、被告
    於YAHOO 奇摩帳號mm66168 申設資料查詢、郵政託運單（流
    水號：000000-000000-00）、著作權授權合約書（英文初級
    文法）、鑑定書（英文初級文法）、被告於PChome露天拍賣
    網站刊登販售謝孟媛Home Run English初級文法之頁面、賣
    家會員資料及評價紀錄等網頁列印資料、被告聯絡交易匯款
    所用之行動電話門號0000000000號查詢資料、臺灣郵政股份
    有限公司中壢郵局97年6 月27日營字第0971100726號函暨所
    檢送之被告帳戶（帳號：000000-0號）開戶基本資料、中華
    郵政股份有限公司97年11月20日儲字第0970086877號函暨所
    檢送之被告帳戶（帳號：000000-0號）開戶資料及歷史交易
    清單、郵政託運單（流水號：000000-000000-00）、郵政國
    內匯款執據、蒐證照片各1 份附卷可稽，是本件事證明確，
    被告自白核與事實相符，其犯嫌已堪認定。
三、核被告甲○○以影印方式重製謝孟媛英文文法教材並加以販
    售之行為，係違反著作權法第91條第2 項之意圖銷售以重製
    之方法侵害他人之語文著作財產權罪；其以燒錄方式重製上
    開教學光碟部分，係犯著作權法第91條第3 項之意圖銷售以
    重製於光碟之方式侵害他人之視聽著作財產權罪；並犯商標
    法第81條第1 款之未經商標權人同意，於同一商品使用相同
    之註冊商標罪。公訴人雖未引用商標法第81條，惟於犯罪事
    實欄已明確記載未經商標權人同意，於同一商品使用相同之
    註冊商標之事實，該部分業經起訴，本院自應加以裁判。按
    著作權法第91條第3 項、第2 項既係以意圖銷售而擅自以重
    製光碟方法侵害他人之著作財產權，為其犯罪成立要件，被
    告意圖散布而持有明知係侵害著作財產權之光碟重製物，當
    為其後散布之高度行為吸收；明知係侵害著作財產權之光碟
    重製物而散布之低度行為，應為意圖銷售而擅自以重製於光
    碟之方法加害他人著作財產權之高度行為所吸收，均不另論
    罪（最高法院92年度台上字第1425號、93年度台上字第950
    號判決意旨參照）。又被告於同一商品使用相同之註冊商標
    後持以販賣，販賣之低度行為，亦為使用相同商標之高度行
    為所吸收，不另論商標法第82條明知為使用相同之註冊商標
    於同一商品而販賣罪。公訴人認被告所為應論商標法第82條
    明知為使用相同商標之同一商品而販賣罪，容有誤會，附此
    說明。再被告利用不知情之影印店人員影印重製盜版教學教
    材，為間接正犯。被告基於同一販賣盜版「謝孟媛英文教學
    DVD ，Home Run English，在家run 英文，中級英文文法」
    、「謝孟媛英文教學DVD ，Home Run English，在家run 英
    文，發音篇」及「謝孟媛英文教學DVD ，Home Run English
    ，在家run 英文，初級英文文法」等教學光碟、教材之決意
    ，自95年年底某日起至為警查獲時止之密集時間內，在上址
    住處上網刊登販賣上開教學教材、教學光碟之訊息，於密切
    接近時間內，以相同之方式持續、反覆以影印及燒錄至光碟
    之方式，持以販賣牟利，未曾間斷，已具有反覆、延續實行
    之特徵，從而，在行為概念上，縱有多次意圖銷售而重製、
    使用相同商標於同一商品之舉措，仍應評價認為係集合犯之
    包括一罪。又被告以同一行為，同時觸犯違反著作權法第91
    條第3 項、第2 項之罪及商標法第81條之罪，及以一意圖銷
    售而重製之行為，同時侵害著作財產權人之語文著作及視聽
    著作之著作權，均為想像競合犯，均依刑法第55條規定，應
    從一重之著作權法第91條第3 項、第2 項意圖銷售而擅自以
    重製於光碟之方法侵害他人之著作財產權罪論處。
四、爰審酌被告為圖利益，任意重製教學教材、光碟，漠視他人
    之著作財產權、商標專用權，實屬不該，惟念及被告係初次
    觸犯著作權法、商標法案件，且坦承犯行，深表悔悟，已與
    告訴人達成和解，有雙方於本院98年6 月10日之準備程序筆
    錄附卷足稽，犯罪後之態度尚佳等一切情狀，量處如主文所
    示之刑。末按緩刑宣告，得斟酌情形，命犯罪行為人向被害
    人支付相當數額之財產或非財產上之損害賠償，刑法第74條
    第2 項第3 款定有明文。查被告未曾受有期徒刑以上刑之宣
    告，有臺灣高等法院被告前案紀錄表可憑，且已與告訴人成
    立和解，賠償告訴人之損害，深具悔意，經此偵、審之教訓
    ，應知所警惕，信無再犯之虞，本院認所宣告之刑以暫不執
    行為適當，依刑法第74條第1 項第1 款規定，併予宣告緩刑
    3 年。而本件被告與告訴人已達成和解，本院參酌其和解條
    件（扣除被告已給付之32萬元後），認於被告緩刑期間課予
    按期還款之負擔，應屬適當，而併宣告命被告應履行如主文
    所示之條件，以啟自新。
五、依刑事訴訟法第449 條第2 項、第3 項、第454 條第1 項，
    著作權法第91條第3 項、第2 項，商標法第81條第1 款，刑
    法第11條前段、第55條，第74條第1 項第1 款、第2 項第3
    款，逕以簡易判決處刑如主文。
六、如不服本判決，得自判決送達後10日內，向本院提起上訴。

附錄本案論罪科刑依據之法條：
著作權法第91條
擅自以重製之方法侵害他人之著作財產權者，處3 年以下有期徒
刑、拘役，或科或併科新臺幣75萬元以下罰金。
意圖銷售或出租而擅自以重製之方法侵害他人之著作財產權者，
處6 月以上5 年以下有期徒刑，得併科新臺幣20萬元以上2 百萬
元以下罰金。
以重製於光碟之方法犯前項之罪者，處6 月以上5 年以下有期徒
刑，得併科新臺幣50萬元以上5 百萬元以下罰金。
著作僅供個人參考或合理使用者，不構成著作權侵害。
商標法第81條
未得商標權人或團體商標權人同意，有下列情形之一者，處3 年
以下有期徒刑、拘役或科或併科新臺幣20萬元以下罰金：
一、於同一商品或服務，使用相同之註冊商標或團體商標者。
二、於類似之商品或服務，使用相同之註冊商標或團體商
    標，有致相關消費者混淆誤認之虞者。
三、於同一或類似之商品或服務，使用近似於其註冊商標
    或團體商標之商標，有致相關消費者混淆誤認之虞者。
